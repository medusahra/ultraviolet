# ULTRAVIOLET
Convertidor simple de imagen a ASCII art con efectos de color ultravioleta.

## Features

- Conversión imagen → ASCII
- 9 modos de color (medusahra signature, gradient, matrix, vaporwave, etc.)
- 8 character sets diferentes
- Export PNG y TXT
- Interfaz minimalista

## Stack

- Vanilla JavaScript
- Canvas API
- HTML/CSS
